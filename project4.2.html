Heading Tags..

How to Add HTML Headings To Your Web Page

In HTML, are written in the following elements:

Levels Of Heading Tags

  • <h1>......</h1>
  • <h2>......</h2>
  • <h3>......</h3>
  • <h4>......</h4>
  • <h5>......</h5>
  • <h6>......</h6>

As you might have guessed < h1 > and < h2 > should be used for the most important titles, while
the remaining tags should be used for sub-headings and less important text.
Search engine bots use this order when deciphering which information is most important on a page
